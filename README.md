# Expenses Tracker

[![CircleCI](https://circleci.com/gh/gihooh/expenses.svg?style=svg)](https://circleci.com/gh/gihooh/expenses)

## Installation + Running
1. cd expenses
2. npm install
3. Open project in  android studio
4. npm run android
